Place your 3D model files (GLB) in this directory.

The game is set up to load models from this directory and its subdirectories.
For weapons, place them in the 'weapons' subdirectory.

Recommended models to add:
- weapons/assault_rifle.glb
- weapons/shotgun.glb
- weapons/pistol.glb
- environment/platform.glb
- environment/obstacle.glb
- characters/player.glb

You can find free 3D models on websites like:
- https://sketchfab.com (free section)
- https://free3d.com
- https://www.turbosquid.com (free section)
- https://www.cgtrader.com (free section)
